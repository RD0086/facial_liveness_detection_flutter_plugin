## 概述
Android/IOS/H5/小程序活体检测 (支持静默，炫彩，眨眼，摇头，点头，张嘴，远近及随意动作组合)，前后端结合，确保活体检测结果安全可靠。(***可用于实人认证，刷脸认证，活体检测等场景***，SDK不联网)
